# 🎓 EduSense - App de Apoio Estudantil

Este projeto é um protótipo de uma plataforma integrada para estudantes, focada em organizar a vida acadêmica e monitorar o bem-estar emocional.

Você pode visualizar o projeto rodando aqui:

---

## 📋 Sobre o Projeto
O **EduSense** foi criado para resolver a falta de conexão entre o desempenho escolar e a saúde mental dos alunos. Ele centraliza informações que normalmente estão espalhadas.

### Funcionalidades atuais (Protótipo):
* **Dashboard:** Resumo de notas e faltas.
* **Check-in de Humor:** Espaço para o aluno registrar como se sente no dia.
* **Apoio e Dicas:** Conteúdo curado para ajudar nos estudos.
* **Projeto de Vida:** Acompanhamento de metas pessoais e competências.

## 🚀 Melhorias Futuras (Roadmap)
Como o projeto está em fase de protótipo, as próximas etapas de desenvolvimento incluem:
1.  **Backend & Banco de Dados:** Implementar sistema de login e salvar os dados de humor e notas de forma persistente.
2.  **Notificações:** Alertas para datas de provas e lembretes de saúde mental.
3.  **Painel do Professor:** Uma visão para que os docentes possam ver estatísticas gerais da turma e identificar alunos que precisam de apoio.
4.  **Gráficos Animados:** Uso de bibliotecas como Chart.js para mostrar a evolução das notas visualmente.
